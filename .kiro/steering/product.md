# Product Overview

## Product Purpose
LinguaLive is a real-time multilingual captioning platform that enables speakers to broadcast live captions to viewers in multiple languages. It solves the accessibility and language barrier problem in classrooms, conferences, meetings, and live events.

## Target Users
- **Speakers/Presenters**: Teachers, lecturers, conference speakers who need to reach multilingual audiences
- **Viewers**: Students, attendees, participants who need captions in their preferred language
- **Event Organizers**: People running multilingual events, webinars, or classes
- **Accessibility Users**: Deaf/hard-of-hearing individuals who rely on captions

## Key Features
- Real-time speech-to-text transcription via Speechmatics
- Live translation to 5 languages (English, Spanish, French, German, Chinese)
- Room-based sessions with shareable codes and QR codes
- AI-powered smart punctuation and grammar correction
- Session recording and history
- Text-to-speech for viewers (Listen mode)
- Keyword highlighting for important terms
- Dark/Light theme support
- Mobile-responsive design

## Business Objectives
- Reduce language barriers in educational settings
- Improve accessibility for hearing-impaired users
- Enable real-time multilingual communication
- Provide affordable captioning alternative to human interpreters

## User Journey
1. **Speaker** opens /speaker, gets a room code
2. **Speaker** shares room code or QR code with viewers
3. **Viewers** join via /viewer, enter room code, select language
4. **Speaker** starts speaking, captions appear in real-time
5. **Viewers** see translated captions in their chosen language
6. **Speaker** can save session to history, export transcript

## Success Criteria
- Caption latency under 2 seconds
- Translation accuracy above 90%
- Support for 50+ concurrent viewers per room
- Mobile usability score above 90
- Accessibility compliance (WCAG 2.1 AA)
