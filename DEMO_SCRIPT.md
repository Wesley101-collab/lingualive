## Recording Tips

### Voice Options (Recommended)
1. **ElevenLabs** - Most natural, supports multiple languages
2. **Murf.ai** - Professional, good for demos
3. **Google Cloud Text-to-Speech** - Free, decent quality
4. **Your own voice** - Also fine if you're comfortable

### Screen Recording
- Use OBS Studio (free) or Loom
- Record at 1920x1080 resolution
- 30 FPS is sufficient
- Enable cursor highlighting

### Preparation Checklist
- [ ] Clear browser cache/cookies for clean demo
- [ ] Prepare test audio or script to speak
- [ ] Have room code ready (or know it generates quickly)
- [ ] Test all features beforehand
- [ ] Close unnecessary browser tabs
- [ ] Disable notifications
- [ ] Use incognito/private mode for clean UI

### Editing
- Add smooth transitions between scenes
- Speed up slow parts (2x) if needed
- Add text overlays for key features
- Background music (optional, keep it subtle)
- Export as MP4, H.264 codec

### Alternative: Quick Version (2 minutes)
If time is tight, combine scenes:
- Intro (15s)
- Speaker + Live Transcription (45s)
- Viewer + Translation (30s)
- AI Features (20s)
- Kiro CLI (20s)
- Closing (10s)
